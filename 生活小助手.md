# 生活小助手

## 技术架构

- Bmob 后端云技术
- RxVolley 网络框架
- Picasso 多图异步加载
- zxing 二维码库
- 百度地图 API
- 科大讯飞 TTS 语音引擎
- 自定义键盘及逻辑
- Tomcat 文件下载更新
- CircleImageView 图形头像
- PhotoView 图片缩放拖拽
- 实现时间轴效果
- 自定义短信提醒库
